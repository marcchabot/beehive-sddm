# Bee-Hive SDDM 🐝🔐

Thème de connexion SDDM pour **Bee-Hive OS**, basé sur **SilentSDDM** (uiriansan).

## Caractéristiques
- **Nectar Sync 🍯** : Adaptation automatique du thème (HoneyDark/HoneyLight).
- **Images Animées** : Support des GIF et vidéos de fond.
- **Palette Bee-Hive** : Couleurs synchronisées avec le reste de l'OS.
